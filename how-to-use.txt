이 프로젝트에는 실제로 두 개의 프로젝트가 포함되어 있습니다:
- - A React.js 애플리케이션(즉, 프론트엔드 SPA)
- - React 앱이 "대화"할 수 있는 더미 백엔드 API(데이터 전송 + 가져오기)

두 프로젝트 폴더 모두에서 "npm install"을 실행해야 합니다.

그 후, "npm start"("backend-api" 폴더 내)를 통해 더미 백엔드 API 서버를 시작할 수 있습니다.
리액트 앱 개발 서버는 ("리액트 프론트엔드" 폴더 내에 있지만) npm 시작을 통해서도 시작된다.

프로젝트가 작동하려면 두 서버(백엔드 + 프론트엔드)가 모두 가동 중이어야 합니다.

더미 백엔드 API는 외부 데이터베이스를 사용하지 않으며, 더미 데이터는 프로젝트 폴더 내의 events.json 파일에 저장된다.
