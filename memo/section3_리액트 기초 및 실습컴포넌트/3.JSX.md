JSX는 Javascript XML을 의미한다.

원래 JS 코드 내에서는 HTML을 사용할 수 없지만,
React 화면 뒷단에서 실행되는 변환 과정이 있기 때문에 JSX라는 HTML을 사용할 수 있게된다.
